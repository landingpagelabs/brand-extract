How it works

Brand Extract fetches the HTML and CSS from any public website, then parses it to find favicons, colour values, and font declarations. Colours are extracted from CSS custom properties, inline styles, and linked stylesheets. Fonts are detected from Google Fonts links, @font-face rules, and font-family declarations. Everything runs through a serverless function — nothing is stored.

What you get

  • Favicons at multiple sizes with download and copy URL options
  • Colour palette with hex, RGB, and HSL values — click to copy
  • Font list with source detection and live previews
  • One-click export as JSON, palette, or a self-contained brand kit HTML file
